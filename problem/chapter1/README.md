# C언어 입문 문제집

입출력, 조건문, 반복문, 함수를 활용한 C언어 실습 문제 10선입니다.

## 컴파일 방법

```bash
gcc problem01_arithmetic.c -o problem01
./problem01
```

---

## 문제 목록

### 문제 1 - 사칙연산 출력 (`problem01_arithmetic.c`)

**핵심 개념:** 입출력, 기본 연산자

두 정수를 입력받아 합, 차, 곱, 몫, 나머지를 출력합니다.

| 입력 | 출력 |
|------|------|
| `10 3` | 합: 13 / 차: 7 / 곱: 30 / 몫: 3 / 나머지: 1 |

**포인트:**
- `scanf`로 두 정수를 동시에 입력받기
- 정수 나눗셈(`/`)과 나머지(`%`) 연산자 구분

---

### 문제 2 - 홀수/짝수 판별 (`problem02_odd_even.c`)

**핵심 개념:** 입출력, 조건문(`if-else`), 나머지 연산자

정수를 입력받아 홀수인지 짝수인지 판별합니다.

| 입력 | 출력 |
|------|------|
| `7` | 7은(는) 홀수입니다. |
| `4` | 4은(는) 짝수입니다. |

**포인트:**
- `n % 2 == 0` 이면 짝수, 아니면 홀수
- 0은 짝수로 처리

---

### 문제 3 - 학점 계산 (`problem03_grade.c`)

**핵심 개념:** 조건문(`if-else if-else`), 범위 검사

0~100 사이의 점수를 입력받아 학점(A~F)을 출력합니다.

| 점수 범위 | 학점 |
|----------|------|
| 90 이상 | A |
| 80 이상 | B |
| 70 이상 | C |
| 60 이상 | D |
| 60 미만 | F |

**포인트:**
- `else if`를 이용한 다중 조건 처리
- 유효 범위(0~100) 검사 먼저 수행

---

### 문제 4 - 구구단 출력 (`problem04_multiplication.c`)

**핵심 개념:** 조건문, `for` 반복문

2~9 사이의 단을 입력받아 해당 구구단을 출력합니다.

```
입력: 3
출력:
3 x 1 = 3
3 x 2 = 6
...
3 x 9 = 27
```

**포인트:**
- `for` 반복문으로 1~9까지 순회
- 잘못된 입력(범위 초과) 처리

---

### 문제 5 - 1부터 N까지의 합 (`problem05_sum_loop.c`)

**핵심 개념:** `for` 반복문, 누적 합산

양의 정수 N을 입력받아 1부터 N까지의 합을 반복문으로 계산합니다.

| 입력 | 출력 |
|------|------|
| `10` | 1부터 10까지의 합: 55 |
| `100` | 1부터 100까지의 합: 5050 |

**포인트:**
- 누적 변수 `sum`을 0으로 초기화 후 반복 덧셈
- 수학 공식(N*(N+1)/2)과 결과를 비교해보기

---

### 문제 6 - 팩토리얼 계산 (`problem06_factorial.c`)

**핵심 개념:** 함수 정의/호출, `for` 반복문, `long long` 타입

0 이상의 정수 N을 입력받아 N! 을 계산하는 **함수**를 작성하고 결과를 출력합니다.

| 입력 | 출력 |
|------|------|
| `5` | 5! = 120 |
| `10` | 10! = 3628800 |

**포인트:**
- 함수 선언(`long long factorial(int n)`)과 호출 분리
- 0! = 1 (기저 조건)
- 큰 값을 담기 위해 `long long` 사용

---

### 문제 7 - 피보나치 수열 출력 (`problem07_fibonacci.c`)

**핵심 개념:** `for` 반복문, 변수 교환, `long long` 타입

N을 입력받아 피보나치 수열의 첫 N개 항을 출력합니다.

```
입력: 8
출력: 1 1 2 3 5 8 13 21
```

**포인트:**
- 이전 두 항을 저장하는 변수 `a`, `b` 활용
- 반복마다 `a = b`, `b = a + b` 순서 주의

---

### 문제 8 - 소수 판별 (`problem08_prime.c`)

**핵심 개념:** 함수, `for` 반복문, 조건문, 최적화

N이 소수인지 판별하고, 2부터 N까지의 모든 소수를 출력합니다.

```
입력: 20
출력:
20은(는) 소수가 아닙니다.
2부터 20까지의 소수: 2 3 5 7 11 13 17 19
```

**포인트:**
- `is_prime(int n)` 함수로 로직 분리
- `i * i <= n` 까지만 나누어 보는 최적화 (√n 이하까지만 확인)

---

### 문제 9 - 배열 통계 (`problem09_array_stats.c`)

**핵심 개념:** 배열, 함수에 배열 전달, 형 변환

N개의 정수를 입력받아 최솟값, 최댓값, 평균을 출력합니다. 각 기능을 별도 함수로 구현합니다.

```
입력:
5
3 1 4 1 5

출력:
최솟값: 1
최댓값: 5
평균: 2.80
```

**포인트:**
- 함수에 배열을 전달할 때 크기 `n`도 함께 전달
- 평균 계산 시 `(double)sum / n` 으로 정수 나눗셈 방지

---

### 문제 10 - 메뉴 기반 계산기 (`problem10_calculator.c`)

**핵심 개념:** `while` 반복문, `switch-case`, 함수, 포인터(출력 매개변수)

반복문과 메뉴를 이용하여 사칙연산 계산기를 구현합니다. 0을 입력하면 종료됩니다.

```
===== 계산기 =====
1. 덧셈
2. 뺄셈
3. 곱셈
4. 나눗셈
0. 종료
선택: 1
두 수를 입력하세요: 10 5
결과: 10.00 + 5.00 = 15.00
```

**포인트:**
- `while(1)` + `break`로 무한 루프 탈출
- 나눗셈 함수에서 0 나누기 방지를 위해 포인터 매개변수 활용
- `switch-case`로 메뉴 분기

## 컴파일 옵션 (권장)

```bash
gcc -Wall -Wextra -std=c11 problem01_arithmetic.c -o problem01
```

- `-Wall -Wextra`: 경고 메시지 최대 출력 (좋은 코드 습관 형성)
- `-std=c11`: C11 표준 사용 (가변 길이 배열 등 지원)
